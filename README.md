# [DeathNote.com.br](https://www.deathnote.com.br)  
Site feito em ReactJS por um fã de Death Note.

## Guia de contribuição:
Sinta-se livre para contribuir, apenas siga a filosofia do projeto:
 - Feito **NÃO** é melhor que perfeito; bem feito, sim.
   - Efeito de sticky scoll foi removido por não funcionar como
   deveria em alguns dispositivos, além de apresentar alguns bugs.
   Sendo assim, é melhor um efeito de scroll comum, mas que não
   irá irritar o usuário devido à usabilidade questionável.  

 - Use a criatividade: Sinta-se livre para expressar sua criatividade,
adicionando novos recursos ou melhorando com existentes.

 - Novos recursos devem, obviamente, ser relacionados com *Death Note*.

 - Não há testes unitários no momento pois, o código TypeScript é
 quase que inteiramente componentes funcionais com praticamente
 nenhuma lógica. Porém, à medida que a lógica e complexidade forem
 aumentando, testes serão adicionados e exigidos.

 - Imagens devem, de preferência, ser otimizadas e comprimidas. 
 *Lazy-loading* deve ser usado quando possível.
